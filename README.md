A real-time, multi-agent web application built to automate and coordinate hackathon operations end-to-end.

### 🧠 PatternIQ — DSA Learning & Retention System
> **React · Vite · Tailwind · Supabase**

A personal DSA preparation system built around structured learning and long-term retention — not just completion tracking.

- **428 LeetCode problems** across **87 DSA patterns**
- **SM-2 Spaced Repetition** for intelligent revision scheduling
- Mistake Journal + Learning Logs
- Progress Analytics Dashboard
- Cloud sync via Supabase with unique user IDs

> *Most DSA trackers tell you what you've done. PatternIQ tells you what you need to revisit — and when.*
